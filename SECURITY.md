# Security Policy

## Supported Versions

Security updates are currently focused on the latest main branch.

## Reporting a Vulnerability

Please do not open a public issue for suspected security vulnerabilities.
Instead, email the maintainers directly with details about the issue and recommended remediation steps.

## Expectations

- Avoid exposing secrets, keys, or tokens in code or issues
- Use environment variables for sensitive configuration
- Report vulnerabilities responsibly and privately
